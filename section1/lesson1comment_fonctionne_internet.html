Module 1 – Leçon 3 : Comment fonctionne Internet ?

1. Définition simple d'Internet

Internet est un réseau physique mondial composé de câbles (notamment en fibre optique) et d’ordinateurs connectés. Il permet l’échange instantané de données entre deux machines, quelle que soit leur distance géographique.

2. Les deux rôles fondamentaux : Client et Server

  • Client : appareil utilisé par l’utilisateur (ordinateur, smartphone...). Il envoie des requêtes.
  • Server : ordinateur spécialisé qui répond aux requêtes et fournit les fichiers demandés.

3. Comment une adresse web (URL) est traduite ?

  1. Le navigateur envoie une requête à l’ISP (fournisseur d’accès).
  2. L’ISP contacte un serveur DNS.
  3. Le serveur DNS traduit le nom de domaine en adresse IP.
  4. Le navigateur utilise cette IP pour contacter le Server.

4. Exemple de scénario complet

  1. Tu tapes google.com dans ton navigateur.
  2. L’ISP demande au DNS l’adresse IP.
  3. Le DNS répond avec 142.250.74.206.
  4. Le navigateur contacte cette IP.
  5. Le Server de Google renvoie les fichiers nécessaires.

5. Les câbles sous-marins et la portée mondiale

Des câbles en fibre optique relient les continents. Chaque câble transporte des données à très haute vitesse (par laser), avec des centaines de fibres pour des centaines de gigabits/s.

6. À retenir sur le fonctionnement d’Internet

  • Internet est un réseau de câbles, pas un nuage.
  • Les données circulent de Client à Server via des requêtes.
  • Le DNS traduit les noms de domaine en IP.
  • L’ISP est le point de départ des requêtes.
  • Les données traversent le monde à la vitesse de la lumière.
  • Un aller-retour d’une requête prend quelques millisecondes.

7. Vocabulaire essentiel

  • Internet : réseau global
  • Client : appareil utilisateur
  • Server : machine qui répond
  • URL : adresse web
  • ISP : Internet Service Provider
  • DNS server : traduit les noms de domaine en IP
  • IP address : identifiant numérique d’un appareil
  • Web server : serveur web
  • Fiber Optic Cable : câble de fibre optique

8. Applications pédagogiques

  • Entre un nom de site dans ton navigateur et mesure la vitesse d’affichage.
  • Utilise nslookup.io pour trouver l’IP d’un nom de domaine.
  • Estime la distance que les données ont parcourue.

9. Transition vers la suite du cours

Maintenant que tu comprends le fonctionnement d’Internet, tu vas apprendre à construire un site web en HTML, puis le styliser avec CSS et le rendre interactif avec JavaScript.
